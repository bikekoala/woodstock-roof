# woodstock-roof

荣威 eRX5 2017 车顶多层伸缩太阳能平台 — DIY 设计项目。

用标准铝型材搭一个**立方体箱体**，竖直分层利用：顶层水箱+固定太阳能板、中层双块可左右伸缩的柔性太阳能板、底层下挂太阳能板，后段开放置物区。目标低矮、受力合理、淘宝下单到家只拧螺丝。

## 快速开始

**看 3D 模型**：浏览器打开 [cad/roof-rack-3d.html](cad/roof-rack-3d.html)
- 拖动旋转 · 滚轮缩放 · 右键平移
- 右上角滑块改尺寸/展开程度，模型实时联动
- 左下角是材料清单 BOM
- 点击任意零件（或清单某行）看它的编号、材料、截面、长度
