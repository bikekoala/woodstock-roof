# 设计规格 — 车顶太阳能平台（方案 A）

> 与 [cad/roof-rack-3d.html](../cad/roof-rack-3d.html) 一致；[cad/model.js](../cad/model.js) 为几何与 BOM 的唯一数据源，本文为离线说明。

## 1. 总体架构

单一**立方体铝型材箱体**，竖直方向分层利用。骨架（底框+4 角立柱+顶框）承力，内部按高度安排太阳能板、水箱、置物。

层级（自上而下）：
1. **顶层**：前半 = 固定太阳能板(顶板) 盖在水箱上方；后半 = 开放置物区(DIY)
2. **吊装托盘**：从顶框往下吊一个整体托盘，前半兜水箱、后半托杂物
3. **中层**：板 A(左滑)、板 B(右滑) 两块柔性板叠放，左右伸缩
4. **底框**：4040 骨架，坐在车顶 OEM 纵向导轨上（U 型卡扣，免打孔）
5. **底层**：板 C 下挂，向前伸缩

| railLength | 主纵梁长 | 1300 | **已实测折中**：支撑段 1000 + 前坡 250/后坡 200 |

**派生（宽度语义见 [decisions.md](decisions.md) D11）：**
- 箱体总外宽 = railSpacing = **1020**（两侧纵梁各 40 已含在内）
- 内腔净宽（横梁 M2/T2 长）= railSpacing − 2×40 = **940**
- 箱体总高 = zBot + 40(底框) + boxHeight + 40(顶框) ≈ **225**（⚠️ 略超 20cm 红线，方案 A 代价）
- 展开总宽 = 1020 + 2×panelTravel = **3020**（⚠️ 行驶前必须收回）
- 滑板 A/B 长边 = railLength − 120 = **1180**，y 向居中：对角立柱净隙 20、对竖撑 S1 净隙 40（解滑出冲突，见 [decisions.md](decisions.md) D9）
- 托盘内缩到立柱内侧，承托横档 D3 = railSpacing − 120 = **900**，水箱同宽（D11，杜绝穿柱）

## 5. 结构与受力要点

- **底框**坐在 OEM 导轨上，载荷直传车顶。水箱须落在前后卡扣之间，勿前探悬臂。
- **顶框 T1 留 4040**：托盘吊在 T1 跨中受弯，4040≈0.7mm vs 2020≈8.9mm；**T2 降 2020**——由中央吊撑 S2 在跨中加支点、跨度减半后挠度 0.4mm 足够（D12）。
- **中央纵梁脊**（M3 底 / D4 托盘）+ **竖撑 S1（下，托盘↔底框）/ 吊撑 S2（上，托盘↔顶框）**：托住跨度中点，替代会挡滑板的斜撑。M3 降 2020（仅中脊/抗扭，不在承重路）。
- **前后端中央立柱 S3**（D4 端部↔T2 端横梁中点，前后各一根）：**非承力件，功能是水平拦阻**——前后端中线在 D3(z135) 与 T2(z205) 之间本有 70mm 高的开口，急刹/颠簸时水箱与置物会从此飞出；S3 把这道口子封成"门框"，兼作绑带/网兜锚点（D16）。
- **抗侧倾**：去斜撑后靠角码刚度——8 个箱角 + 连车点用 L 加强（带筋）角码。
- **滑板悬臂**：A/B 单侧伸出 1m，柔性板自重小，主要防风载与振动。
- **纵向落位与渐缩导轨**：导轨平直支撑段 1000，箱体 1300 → 前后各悬挑 ~150（前坡 250/后坡 200 范围内），4040 悬挑挠度可忽略。导轨前宽后窄（外缘跨距 1070→1020 = 箱体总外宽），U 卡扣现场对位即可。
- **实体干涉自检**：`scripts/check-interference.js` 对全部零件全行程扫描，非法穿插必须为 0，已挂 pre-commit。见 [decisions.md](decisions.md) D10/D11。
