# 设计规格 — 车顶太阳能平台（方案 A）

> 与 [cad/roof-rack-3d.html](../cad/roof-rack-3d.html) 一致；HTML 为几何与 BOM 的真实来源，本文为离线说明。

## 1. 总体架构

单一**立方体铝型材箱体**，竖直方向分层利用。骨架（底框+4 角立柱+顶框）承力，内部按高度安排太阳能板、水箱、置物。

层级（自上而下）：
1. **顶层**：前半 = 固定太阳能板(顶板) 盖在水箱上方；后半 = 开放置物区(DIY)
2. **吊装托盘**：从顶框往下吊一个整体托盘，前半兜水箱、后半托杂物
3. **中层**：板 A(左滑)、板 B(右滑) 两块柔性板叠放，左右伸缩
4. **底框**：4040 骨架，坐在车顶 OEM 纵向导轨上（U 型卡扣，免打孔）
5. **底层**：板 C 下挂，向前伸缩

## 2. 尺寸链 / 主控参数

| 参数 | 含义 | 默认(mm) | 来源 |
|---|---|---|---|
| railSpacing | OEM 导轨中心距 | 1000 | **实测** |
| railLength | 主纵梁长 | 1200 | 可调 |
| boxHeight | 箱体内净高(立柱高) | 110 | 可调 |
| panelTravel | 单侧抽出行程 | 1000 | 可调 |
| tankH | 水箱/托盘深 | 70 | 定制 |
| zBot | OEM 导轨抬高(底框底面) | 35 | **实测** |
| (固定) | 主型材 4040 / 轻型材 2020 / 板 1000×1200 | | |

**派生：**
- 箱体外宽 = railSpacing + 2×40 ≈ **1080**
- 箱体总高 = zBot + 40(底框) + boxHeight + 40(顶框) ≈ **225**（⚠️ 略超 20cm 红线，方案 A 代价）
- 展开总宽 = 1080 + 2×panelTravel ≈ **3080**（⚠️ 行驶前必须收回）

## 5. 结构与受力要点

- **底框**坐在 OEM 导轨上，载荷直传车顶。水箱须落在前后卡扣之间，勿前探悬臂。
- **顶框升 4040**：水箱吊在顶框下受弯；2020 动载下垂 ~20mm，4040 仅 ~2mm。
- **中央纵梁脊**（M3 底 / D4 托盘）+ **中央竖撑 S1**：托住跨度中点，替代会挡滑板的斜撑。
- **抗侧倾**：去斜撑后靠角码刚度——8 个箱角 + 连车点用 L 加强（带筋）角码。
- **滑板悬臂**：A/B 单侧伸出 1m，柔性板自重小，主要防风载与振动。

## 6. 太阳能汇总（参考原始目标）

| 板 | 位置 | 尺寸 | 动作 |
|---|---|---|---|
| 板A | 中层上 | 1000×1200 | 左抽 |
| 板B | 中层下 | 1000×1200 | 右抽 |
| 板C | 底框下挂 | 1000×1200 | 前抽 |
| 顶板 | 顶框前半 | 1000×~540 | 固定 |

目标发电面积 ~3.5㎡、峰值 ~700W（ETFE 23%）。
